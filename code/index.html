• Migratory Goose

Bienvenue sur la page du projet Migratory Goose

Description

Le projet Migratory Goose est un petit système multi-agent sur le thème de la migration des oiseaux.
Les agents de ce système sont des populations d’oiseaux, ayant des comportements migratoires ou sédentaires. Elles évoluent sur une carte contenant des éléments influençant leur évolution.
A travers cette simulation, vous allez pouvoir observer les populations d'oiseaux progresser sur la carte et suivre en temps réel leurs évolutions grâce à l'affiche de graphiques.

Comment utiliser cette simulation ?

Le nombre d’arbres influence le déplacement des oiseaux. Il est plus difficile pour eux de voler à travers les cimes et ils auront tendance à préférer les zones dégagées. Le nombre d’arbres doit être compris entre 0 et 1000.
Le nombre initial d'individus dans chaque population de la simulation doit être compris entre 1 et 100.
Les taux de nourriture, de catastrophes naturelles et de reproduction doivent êtres compris entre 0 et 1. Attention, la somme des taux de nourriture et de catastrophes naturelles ne peut pas dépasser 1.
Une fois les paramètres choisis, vous pouvez cliquer sur Confirmer les paramètres, puis sur le bouton Lancer la simulation. Vous pouvez également utiliser le bouton Réinitialiser les paramètres pour les faire revenir à leurs valeurs par défaut.
Les boutons Scénario 1, 2 et 3 correspondent à des jeux de paramètres précis, utilisés pour comparer différents environnements de simulation. Les résultats de ces tests sont disponibles dans la partie Scénario de tests du rapport associé au projet.
Une fois la simulation lancée, les paramètres ne peuvent plus être modifiés. Les boutons Lancer la simulation et Arrêter la simulation permettent de faire des pauses à n’importe quel moment et de reprendre ensuite la simulation là où elle s’est arrêtée.

La documentation complète du projet, ainsi que le code source sont disponibles sur la page github MigratoryGoose.
